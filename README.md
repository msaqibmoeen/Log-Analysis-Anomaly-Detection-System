# Log Analysis & Anomaly Detection System

## Overview

This project was developed as part of the **Machine Learning Lab** course during my 6th semester at **COMSATS University Islamabad, Sahiwal Campus**, under the supervision of **Mam Raheela Shahzadi**.

The system is an **AIOps-based Log Analysis & Anomaly Detection Platform** designed to monitor system logs, detect anomalies, classify log severity, and generate intelligent alerts using Machine Learning techniques. The project integrates multiple ML paradigms including supervised learning, unsupervised learning, optimization, ensemble learning, and reinforcement learning into a single full-stack application.

---

## Features

* Real-time log classification
* Anomaly detection and alert generation
* Intelligent alert system (Normal / Warning / Critical)
* Adaptive threshold learning using Reinforcement Learning
* Model comparison dashboard
* Live log stream visualization
* Confusion Matrix and evaluation metrics
* Interactive analytics dashboard
* Modern web-based user interface
